A BOOK-MARKING WEB APPLICATION.

This entry features a book-marking django web application which allows users the ability to like 'Categories' and add 'Pages' to specific select categories. Python is used alongside JavaScript, JQuery, and AJAX in order to implement and simulate dynamic features.
Authenticated users can freely interact with all features of the site which include but not limited to:- profile accounts with enabled profile-image uploads, personal website entries, viewing of other user profiles, and bio-message displays.
Styling was accomplished by custom CSS and Bootstrap4 application. Testing and debugging was done prior to deployment of the application online.
The web application is fully-functional, readily accessible online, and goes by the name 'Page-House'. The copyright seal is also denoted on the About-page.
