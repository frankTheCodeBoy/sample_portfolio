THE LEARNING-LOG PROJECT.

This project is again a live web application built with Python and Django framework. The application serves as an online journal for daily activity entry. Users register and create accounts to actively use it. The application features topics; public and private topics, and entries which are added to topics created. Users track their activities and learning progress by logging new topics and updating it with new ideas which appear in form of entries unique to owners.
Prior to deployment, necessary tests were conducted and git-tracking was enabled consistently. Code was finally configured for production after the necessary migrations were conducted locally and on the server. This application which is named 'Learning Log', is currently active and can be directly accessed through this link.

Point of note: depending on your network strength or location, the link to the web application may take roughly 5 - 7 seconds to load up at first. At the least, give it this minimum time limit.
