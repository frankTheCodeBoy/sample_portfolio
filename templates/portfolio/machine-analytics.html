MACHINE LEARNING DATA ANALYTICS.

Creating machine algorithms and models that work well, usually involves combining the processess of data visualisation and data analysis. In this section, Python library and modules were used to mould machine models from validated and tested algorithms. Here I have included illustrations of some models that were trained and tested on various datasets involving both Supervised and Unsupervised learning problems.
The first illustration shown below applies simple linear regression toward solving a supervised problem. The dataset was a CSV format data containing varying salaries for various years in experience. The dataset was split into train and test sets using the train-test-split function from the Sklearn module. The dataset was fitted into the training set and testing set, and visualisations drawn from these. It performed well as shown in the charts below (the same performance was depicted when the model was tested on new data).
The complete code for this analysis can be downloaded from this repository.

The next block of diagrams demonstrate the implementation of K-Nearest Neighbors in creating a model that solves Classification problem under supervised learning. Initially, the K-Neighbors Classifier and the Confusion Matrix were imported from the Sklearn module. Feature scaling was applied to normalise the dataset after obtaining the train and test splits from the data. The dataset used was CSV data containing the social status of different gender. The overall objective being to classify different individuals according to their spending pattern (either a '1' or '0') and tabulated salary.
The model's performance was fairly good (owing to the fact that no model or data can achieve 100 percent perfection). Slightly increasing the dataset size for our model could yield better results with less anomalies. Below is the output obtained from the visualisations of train and test sets. My dataset and codes for this work can still be accessed from this repository.

This last section seeks to illustrate the development of an algorithm and a model to resolve unsupervised clustering problem. We examine a CSV data containing the spending scales of mall customers of different income levels. The machine model is developed in an attempt to help cluster and relevantly group customers which in turn helps to tailor marketing strategies effectively for each study group. The WCSS and the Elbow method was used to obtain a suitable cluster number for this analysis. Data was eventually plotted after pre-requisites were met.
