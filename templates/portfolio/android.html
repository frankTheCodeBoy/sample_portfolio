AN ANDROID APPLICATION.

Programming languages have their own uniqueness in terms of practical adaptability towards completing and executing task of varying nature. Java programming language is usually a convinient choice for creating android applications. As such, I have developed a simple but fully-functional 'Number Guessing Game' as an android application written in Java.
The application translates user-input and gives feedback, stores user-settings, preferences, and tracks user-scores. With the collective technologies of Eclipse and Android Studio, I cross-tested it across multiple-device simulations and also deployed it in my own android device. The apk for this application is accessible from this link. It can be readily installed and run on any available android device.
